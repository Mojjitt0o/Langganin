📈 Cara Kerja Profit

1. User order di web kamu dengan harga jual (sell_price)
2. Saldo user dipotong sebesar harga jual
3. Web kamu otomatis order ke Warung Rebahan API dengan harga modal
4. Profit = Harga Jual − Harga Modal → otomatis masuk tabel profits

Contoh: Modal WR Rp 5.000 → kamu jual Rp 6.500 → Profit kamu Rp 1.500 ✅

⚙️ Konfigurasi Markup Otomatis

Ditambahkan dari harga modal WR. Default: 20%

Tambahan fixed per variant. Default: Rp 1.000

⚠️ Pengaturan ini ada di file .env (MARKUP_PERCENTAGE dan FIXED_MARKUP). Ubah di sana untuk perubahan permanen lalu restart server.

💡 Tips Markup yang Bagus

  • Produk murah (< Rp 10.000): markup 30-50% + fixed Rp 500
  • Produk menengah (Rp 10-50k): markup 15-25% + fixed Rp 1.000
  • Produk mahal (> Rp 50.000): markup 10-15% + fixed Rp 2.000
  • Gunakan custom_price per variant untuk kontrol penuh
