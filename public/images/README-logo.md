# Logo Langganin

## Upload Instruksi

Silakan upload logo Anda dengan nama **`logo.png`** di folder ini (`public/images/`).

### Spesifikasi Logo:
- **Format**: PNG dengan background transparan (recommended) atau JPG
- **Ukuran**: Minimal 200px x 200px (square/landscape)
- **Ratio**: 1:1 atau 16:9
- **File name**: `logo.png`

### Lokasi:
```
public/images/logo.png
```

Setelah upload, logo akan otomatis muncul di:
- ✅ Navbar semua halaman
- ✅ Halaman Login & Register
- ✅ Admin Panel

### Fallback:
Jika file belum ada, akan muncul emoji 🚀 sebagai placeholder.
